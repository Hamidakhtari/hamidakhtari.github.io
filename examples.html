Practical HTML Examples

Explore real-world examples of HTML usage:

Example 1: Creating a Basic Web Page

                    
                        <!DOCTYPE html>
                        <html lang="en">
                        <head>
                            <meta charset="UTF-8">
                            <meta name="viewport" content="width=device-width, initial-scale=1.0">
                            <title>My First Web Page</title>
                        </head>
                        <body>
                            <h1>Welcome to My First Web Page</h1>
                            <p>This is a simple web page created using HTML.</p>
                            <a href="https://www.example.com">Visit Example.com</a>
                        </body>
                        </html>

Example 2: Creating an Unordered List

                    
                        <ul>
                            <li>Item 1</li>
                            <li>Item 2</li>
                            <li>Item 3</li>
                        </ul>

Example 3: Adding an Image

                    
                        <img src="example.jpg" alt="Example Image">

Practical CSS Examples

Explore real-world examples of CSS styling:

Example 1: Styling Text

                    
                        p {
                            color: #333;
                            font-size: 1.2em;
                            margin: 10px;
                        }

Example 2: Styling a Button

                    
                        button {
                            background-color: #4CAF50;
                            color: white;
                            padding: 10px;
                            border: none;
                            border-radius: 5px;
                            cursor: pointer;
                        }

Example 3: Creating a Box with Border

                    
                        .box {
                            width: 200px;
                            height: 150px;
                            background-color: #f0f0f0;
                            border: 2px solid #333;
                            padding: 10px;
                            margin: 20px;
                        }
